# Backend

Para ejecutarlo, necesitas configurar la base de datos. Para lograr esto debe crear un archivo
`database.properties` e insertar/modificar el siguiente contenido
```properties
db.url = localhost
db.username = dovah
db.password = dovah
db.database = dovah
```
